HTML Structure

The structure of the html has been kept as minimal as possible to allow you to style your work as freely as possible. Each photo links to the instgram page, but the a tags have a data attritute that you can use reference the larger image if you wanted to lightbox it or something similar. 


		<div id="ins">
			<a data-standard-resolution="URL TO STANDARD RESOLUTION" href="URL TO INSTAGRAM PAGE" rel="instafeed-instagram-photo" title="INSTAGRAM CAPTION">
				<img src="URL TO THUMBNAIL" alt="INSTAGRAM CAPTION">
			</a>
		</div>
